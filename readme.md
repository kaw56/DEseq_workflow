Preparing raw RNAseq data for analysis using DEseq (using Bowtie 2)
===================================================================

Trying to develop a good circadian RNA seq data work flow I'm going
to try DEseq for this one but I think there will be several...

Summary of workflow
===================

Unzip

Fastqc

Bowtie 2

SAM to BAM

Sort the BAM file

index sorted

make a count table

Give to DEseq

Output report
